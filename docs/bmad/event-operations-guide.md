# Event Operations Guide

## Launch Checklist
1. Confirm agenda & speakers.
2. Set pricing/tiers and capacity limits in the Event Dashboard.
3. Enable filters for status/type to monitor performance by cohort.

## Moderation
- Remove cancelled events promptly.
- Use the "Next event" spotlight to brief execs before go-live.

## Post-Event
- Export registrations and revenue summaries.
- Archive or clone successful events for future cohorts.

Use this guide when referencing the in-app Event Dashboard help tooltip.
