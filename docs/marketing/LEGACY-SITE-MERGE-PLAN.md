# Legacy FinanceFlo Site Merge Plan

**Date**: 2025-11-22
**Legacy Site**: https://flo-finance-uk-website.onrender.com/
**New Site**: https://financeflo.ai (ma-saas-platform.onrender.com)
**Status**: Planning

---

## Objective

Merge valuable content, features, and assets from the legacy FinanceFlo website into the new unified platform while maintaining the new site's architecture and design system.

---

## Analysis Required

### 1. Content Audit (30 min)
**Tasks:**
- [ ] Visit legacy site and document all pages
- [ ] Identify unique content not present in new site
- [ ] List valuable assets (images, PDFs, documents)
- [ ] Document any features/functionality to preserve
- [ ] Note any SEO-valuable content (blog posts, case studies)

**Deliverable**: `docs/marketing/legacy-site-content-audit.md`

### 2. Content Mapping (30 min)
**Tasks:**
- [ ] Map legacy pages to new site structure
- [ ] Identify content gaps (what's in legacy but not new)
- [ ] Identify content to migrate vs. archive
- [ ] Note any conflicting information

**Deliverable**: Content mapping document

### 3. Migration Strategy (30 min)
**Tasks:**
- [ ] Prioritize content by value (SEO, user value, uniqueness)
- [ ] Plan migration approach (manual copy vs. automated)
- [ ] Identify any code/features to port
- [ ] Plan redirect strategy for legacy URLs

**Deliverable**: Migration execution plan

## Execution Steps

1. **Content Audit** - Document what exists on legacy site
2. **Gap Analysis** - Compare with new site content
3. **Migration** - Copy valuable content to new site
4. **Redirects** - Set up 301 redirects from legacy URLs
5. **Verification** - Ensure all content migrated correctly
6. **Decommission** - Archive or remove legacy site
