## SEO Best Practices

### Title Optimization
- Include primary keyword near the beginning
- Keep under 60 characters
- Make it compelling and click-worthy

### Meta Description
- Include primary keyword naturally
- Create compelling copy that encourages clicks
- Keep between 150-160 characters

### Content Optimization
- Use H2 headings for main sections
- Include primary keyword in first paragraph
- Use secondary keywords naturally throughout
- Include internal links to related content
- Add external links to authoritative sources

### Image Optimization
- Use descriptive filenames (e.g., `cash-forecast-dashboard.png`)
- Include alt text with keywords
- Optimize file size (compress images)
- Use appropriate dimensions (1200x630px for featured)
