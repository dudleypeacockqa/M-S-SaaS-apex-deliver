## Core Features Guide

### Deal Pipeline Management

**Access**: Navigate to `/deals` from the main menu

**Features**:
- **Kanban View**: Drag and drop deals between stages
- **List View**: Sortable table with filters
- **Deal Details**: Click any deal to view full details
- **Create Deal**: Click "New Deal" button

**Pipeline Stages** (Customizable):
- Sourcing
- Evaluation
- Due Diligence
- Negotiation
- Closing
- Post-Deal

**Actions**:
- Edit deal details
- Add notes and attachments
- Assign team members
- Track deal value and timeline

---

### Financial Intelligence

**Access**: Navigate to `/financial` or `/deals/:id/financial`

**Features**:
- **47+ Financial Ratios**: Automatically calculated from accounting data
- **AI Narratives**: GPT-4 generated insights
- **Deal Readiness Score**: Overall assessment
- **Trend Analysis**: Historical performance tracking

**Integrations**:
- Sage Intacct
- Odoo
- CSV import

**Using Ratios**:
1. Connect your accounting platform
2. Import financial statements
3. View calculated ratios
4. Read AI-generated narratives
5. Export reports

---

### Valuations

**Access**: Navigate to `/deals/:id/valuations` or `/valuations`

**Methods Available**:
1. **DCF (Discounted Cash Flow)**
   - Enter cash flow projections
   - Set discount rate
   - Calculate present value

2. **Comparables Analysis**
   - Select comparable companies
   - Apply valuation multiples
   - Calculate implied value

3. **Precedent Transactions**
   - Review similar deals
   - Apply transaction multiples
   - Estimate deal value

**Creating a Valuation**:
1. Navigate to deal details
2. Click "Create Valuation"
3. Select valuation method
4. Enter assumptions
5. Generate valuation
6. Save and compare

---

### Data Rooms

**Access**: Navigate to `/deals/:id/data-room`

**Features**:
- **Folder Organization**: Create folder hierarchies
- **Permission Groups**: Control access by role
- **Watermarking**: Automatic document watermarking
- **Access Logs**: Track all document views
- **Version Control**: Maintain document versions

**Setting Up a Data Room**:
1. Navigate to deal details
2. Click "Data Room" tab
3. Create folders
4. Upload documents
5. Set permissions
6. Invite external parties

### Task Management

**Access**: Navigate to `/tasks`

**Features**:
- **Task Board**: Kanban view of all tasks
- **Templates**: Reusable task templates

- **Automations**: Trigger actions on completion

**Creating Tasks**:
1. Click "New Task"
2. Enter task details
3. Assign to team member
4. Set due date
5. Add dependencies (optional)
6. Save

## Troubleshooting

### Common Issues

**Issue**: Can't access a deal
**Solution**: Check you have the correct role and organization membership

**Issue**: Financial ratios not calculating
**Solution**: Verify accounting platform connection and data import

**Issue**: Can't upload documents
**Solution**: Check file size limits and permissions

**Issue**: Valuation not generating
**Solution**: Ensure all required assumptions are entered

**Issue**: Tasks not appearing
**Solution**: Check filters and assigned organization
